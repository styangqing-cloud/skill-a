# 冰山模型 - 访谈结果分析指南

## 📖 分析定位

本指南用于**分析面试/评估访谈结果**，通过冰山模型的六层框架评估候选人/员工的胜任力水平，重点挖掘「水面以下」的隐藏素质。

---

## 🎯 分析框架

### 六层分析模型

```
                    ┌─────────────────┐
                    │   水面以上      │
                    │   (可见 20%)    │
                    ├─────────────────┤
                    │ ① 知识          │  ← 容易评估，容易培养
                    │ ② 技能          │
                    ├═════════════════┤ ← 水面线 ═══════
                    │ ③ 社会角色      │
        水面以下    │ ④ 自我认知      │  ← 难以评估，难以改变
        (隐藏 80%)  │ ⑤ 特质          │    决定长期绩效！
                    │ ⑥ 动机          │
                    └─────────────────┘
```

### 分析步骤

#### Step 1: 逐层证据提取

从访谈记录中为每个层次提取证据：

**① 知识层 — 专业知识储备**

| 提取方向 | 证据来源 | 评估标准 |
|---------|---------|---------|
| 专业领域知识 | 对专业问题的回答质量 | 深度/广度/准确度 |
| 行业认知 | 对行业趋势的理解 | 是否有独立见解 |
| 跨领域知识 | 对相关领域的了解 | 知识面的宽度 |

**② 技能层 — 实际操作能力**

| 提取方向 | 证据来源 | 评估标准 |
|---------|---------|---------|
| 核心技能 | 过往项目的技术方案 | 方案的合理性和创新性 |
| 问题解决 | 面对问题的处理方式 | 方法论/系统性/效果 |
| 沟通表达 | 访谈中的表达方式 | 清晰度/逻辑性/说服力 |

**③ 社会角色 — 职业身份认知**

| 提取方向 | 证据来源 | 评估标准 |
|---------|---------|---------|
| 角色定位 | 如何描述自己的职业角色 | 是否清晰/是否与岗位匹配 |
| 责任边界 | 对工作范围的理解 | 是否局限/是否过度扩展 |
| 职业认同 | 对职业的态度和投入 | 认同度/承诺度 |

**④ 自我认知 — 对自身的了解**

| 提取方向 | 证据来源 | 评估标准 |
|---------|---------|---------|
| 优势认知 | 如何描述自己的优势 | 是否客观/有无盲区 |
| 短板认知 | 如何面对自己的不足 | 是否坦诚/是否有改进行动 |
| 反馈接受 | 对他人评价的态度 | 开放度/防御度 |

**⑤ 特质层 — 人格特质和行为倾向**

| 提取方向 | 证据来源 | 评估标准 |
|---------|---------|---------|
| 行为一致性 | 多个事例中的行为模式 | 是否稳定一致 |
| 压力反应 | 面对困难时的表现 | 韧性/应对策略 |
| 人际模式 | 与他人互动的方式 | 协作性/主导性/适应性 |

**⑥ 动机层 — 内在驱动力**

| 提取方向 | 证据来源 | 评估标准 |
|---------|---------|---------|
| 选择模式 | 过往职业选择的核心考量 | 什么驱动了关键选择 |
| 投入模式 | 什么时候最投入 | 兴趣点/热情来源 |
| 满足来源 | 什么带来成就感 | 内在vs外在动机 |

#### Step 2: 证据质量评估

区分证据的可靠性：

| 证据类型 | 可靠度 | 说明 |
|---------|-------|------|
| **行为事例** | ⭐⭐⭐⭐⭐ | 具体的过往行为（STAR格式）最可靠 |
| **具体描述** | ⭐⭐⭐⭐ | 有细节的描述，较可靠 |
| **一般性陈述** | ⭐⭐⭐ | "我比较擅长沟通"——需要事例佐证 |
| **自我评价** | ⭐⭐ | 可能有社会期望偏差 |
| **推测/假设** | ⭐ | "如果遇到...我会..."——参考价值低 |

#### Step 3: 岗位匹配度分析

将提取的六层证据与目标岗位要求对比：

```
岗位匹配评估表：
┌──────────┬─────────────┬─────────────┬──────────────┐
│  层次     │ 岗位要求     │ 评估结果     │ 匹配度      │
├──────────┼─────────────┼─────────────┼──────────────┤
│ ① 知识   │ [要求]      │ [评估]      │ 🟢/🟡/🔴   │
│ ② 技能   │ [要求]      │ [评估]      │ 🟢/🟡/🔴   │
│ ③ 角色   │ [要求]      │ [评估]      │ 🟢/🟡/🔴   │
│ ④ 认知   │ [要求]      │ [评估]      │ 🟢/🟡/🔴   │
│ ⑤ 特质   │ [要求]      │ [评估]      │ 🟢/🟡/🔴   │
│ ⑥ 动机   │ [要求]      │ [评估]      │ 🟢/🟡/🔴   │
└──────────┴─────────────┴─────────────┴──────────────┘
```

#### Step 4: 水面以下风险评估

**重点关注③④⑤⑥层的红旗信号：**

| 层次 | 红旗信号 | 风险 |
|------|---------|------|
| **社会角色** | 角色定位与岗位严重不匹配 | 入职后角色冲突 |
| **自我认知** | 严重高估或低估自己 | 绩效预期偏差 |
| **特质** | 核心特质与岗位要求矛盾 | 长期适应困难 |
| **动机** | 动机与岗位价值不匹配 | 快速流失风险 |

#### Step 5: 发展潜力评估

基于冰山各层评估发展潜力：

| 层次 | 当前水平 | 可培养性 | 发展建议 |
|------|---------|---------|---------|
| 知识 | ... | ⭐ 高 | 培训/学习 |
| 技能 | ... | ⭐ 较高 | 实践/教练 |
| 角色 | ... | ⭐⭐ 中等 | 引导/反馈 |
| 认知 | ... | ⭐⭐⭐ 较低 | 教练/反馈 |
| 特质 | ... | ⭐⭐⭐⭐ 极低 | 适配为主 |
| 动机 | ... | ⭐⭐⭐⭐ 极低 | 激发为主 |

## ⚠️ 分析注意事项

1. **水面以下是关键**：知识和技能可以培养，但特质和动机很难改变
2. **行为证据优先**：自我评价可靠度低，要看具体行为事例
3. **避免晕轮效应**：不要因为某层优秀就推测其他层也优秀
4. **文化适配很重要**：社会角色层要考虑组织文化匹配
5. **动机是最深层驱动**：动机与岗位不匹配是最大的长期风险
