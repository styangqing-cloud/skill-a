# 推拉力理论 - 访谈结果分析指南

## 📖 分析定位

本指南用于**分析离职/挽留访谈结果**，通过推拉力理论的四维度模型系统评估离职决策的驱动力，并量化评估挽留可能性。

---

## 🎯 分析框架

### 四力分析模型

```
┌─────────────────────────────────────────────────────┐
│                  离职决策力量分析                      │
│                                                      │
│    推力 (Push) ───────→  ←─────── 拉力 (Pull)        │
│    促使离开的力量         吸引离开的力量              │
│                                                      │
│    留任力 (Stay) ←────→ 移动障碍 (Barrier)           │
│    留下来的力量           离开的阻力                  │
│                                                      │
│  公式: 推力 + 拉力 > 留任力 + 移动障碍 → 离职        │
└─────────────────────────────────────────────────────┘
```

### 分析步骤

#### Step 1: 四力提取

从访谈记录中逐条提取四类力量的证据：

**推力识别（促使离开的负面因素）：**

| 推力类型 | 识别信号 | 强度判断 |
|---------|---------|---------|
| **工作不满** | "没意思"、"太枯燥"、"不喜欢" | 看持续时间和情绪强度 |
| **发展受限** | "看不到希望"、"晋升无望"、"天花板" | 看是否已尝试过解决 |
| **管理问题** | "leader不行"、"不被信任"、"微管理" | 看是否为核心推力 |
| **薪酬不公** | "薪资倒挂"、"付出与回报不成正比" | 看绝对值还是公平感 |
| **文化不适** | "格格不入"、"加班文化"、"不认同" | 看是否为根本性矛盾 |
| **工作负荷** | "太累了"、"burnout"、"没有life" | 看是否有持续恶化趋势 |
| **技术栈/方向** | "技术太老了"、"方向不看好" | 看对职业发展的影响 |

**拉力识别（吸引离开的外部因素）：**

| 拉力类型 | 识别信号 | 强度判断 |
|---------|---------|---------|
| **更好薪酬** | "外面翻倍"、"明显高于现在" | 看差距大小 |
| **更好发展** | "更大的平台"、"更好的方向" | 看是否有具体机会 |
| **更好文化** | "氛围更好"、"更扁平" | 看体验来源（传闻vs实际） |
| **更好平衡** | "不用加班"、"更自由" | 看优先级高低 |
| **新挑战** | "更有意思的事"、"想尝试新方向" | 看是否已有明确目标 |
| **行业机会** | "风口"、"赛道更好" | 看市场判断的合理性 |

**留任力识别（留下的积极因素）：**

| 留任力类型 | 识别信号 | 强度判断 |
|-----------|---------|---------|
| **情感承诺** | "还是有感情"、"团队很好" | 看情感投入程度 |
| **经济依赖** | "期权/RSU还没vest"、"房贷压力" | 看经济锁定力度 |
| **发展潜力** | "如果能解决...我愿意留" | 看条件是否可满足 |
| **人际关系** | "同事很好"、"不舍得" | 看社交网络深度 |
| **项目牵挂** | "项目还没做完"、"有责任感" | 看时间窗口 |
| **企业认同** | "还是认可公司的"、"品牌好" | 看认同的根基 |

**移动障碍识别（离开的阻力）：**

| 障碍类型 | 识别信号 | 强度判断 |
|---------|---------|---------|
| **转换成本** | "搬家"、"适应期"、"不确定" | 看可克服程度 |
| **家庭因素** | "家人"、"孩子上学"、"配偶工作" | 看家庭决策权重 |
| **地理限制** | "不想换城市"、"户口" | 看是否为硬约束 |
| **竞业限制** | "竞业协议"、"限制行业" | 看法律约束力度 |
| **风险规避** | "外面也不一定好"、"怕后悔" | 看风险承受能力 |
| **市场环境** | "现在行情不好"、"机会不多" | 看客观市场情况 |

#### Step 2: 力量强度评估

对每个力量项评分（1-5分）并汇总：

```
力量评估表：
┌────────────┬───────────────────────┬──────┬────────┐
│ 力量类型    │ 具体因素               │ 强度 │ 证据   │
├────────────┼───────────────────────┼──────┼────────┤
│ 推力       │ 发展受限               │ 5/5  │ "..."  │
│            │ 薪酬不满               │ 3/5  │ "..."  │
│            │ 工作负荷过重           │ 4/5  │ "..."  │
│ 推力综合    │                       │ 4.0  │        │
├────────────┼───────────────────────┼──────┼────────┤
│ 拉力       │ 更好的发展平台         │ 4/5  │ "..."  │
│            │ 更高薪酬               │ 3/5  │ "..."  │
│ 拉力综合    │                       │ 3.5  │        │
├────────────┼───────────────────────┼──────┼────────┤
│ 留任力     │ 团队关系好             │ 3/5  │ "..."  │
│            │ RSU 未 vest            │ 4/5  │ "..."  │
│ 留任力综合  │                       │ 3.5  │        │
├────────────┼───────────────────────┼──────┼────────┤
│ 移动障碍   │ 竞业限制               │ 2/5  │ "..."  │
│ 障碍综合    │                       │ 2.0  │        │
└────────────┴───────────────────────┴──────┴────────┘
```

#### Step 3: 挽留可能性评估

**挽留可能性矩阵：**

| 维度 | 🔴 低（难挽留） | 🟡 中（可尝试挽留） | 🟢 高（大概率可挽留） |
|------|---------------|-------------------|---------------------|
| **推力** | 多个强推力且长期积累（综合≥4） | 1-2个中等推力可针对性解决（2-4） | 推力弱或已在改善（≤2） |
| **拉力** | 已拿到明确offer（综合≥4） | 在看但未确定（2-4） | 未主动寻找（≤2） |
| **留任力** | 几乎无留任因素（综合≤2） | 有一定留任因素（2-4） | 多个强留任（≥4） |
| **障碍** | 障碍极低（综合≤2） | 有一定障碍（2-4） | 障碍较高（≥4） |

**综合判定：**

| 组合 | 挽留可能性 | 建议策略 |
|------|-----------|---------|
| 推力强+拉力强+留任弱+障碍低 | 🔴 **极低** | 做好交接计划，了解改进建议 |
| 推力强+拉力中+留任中+障碍中 | 🟡 **中等** | 快速响应核心推力，展示发展诚意 |
| 推力中+拉力弱+留任强+障碍高 | 🟢 **较高** | 针对性解决推力，强化留任因素 |
| 推力弱+拉力弱+留任强+障碍高 | 🟢 **很高** | 及时干预，可能仅需小调整 |

#### Step 4: 挽留方案设计（如评估为可挽留）

针对核心推力设计挽留方案：

| 推力类型 | 可能的挽留手段 | 时效性 |
|---------|-------------|-------|
| **发展受限** | 岗位轮换/项目机会/导师/晋升通道 | 中期见效 |
| **薪酬不公** | 调薪/奖金/股票/特殊激励 | 短期见效 |
| **管理问题** | 换组/换leader/管理者辅导 | 需谨慎评估 |
| **工作负荷** | 人力补充/优先级调整/灵活工作 | 短期见效 |
| **文化不适** | 深入了解具体不适点/部门文化差异 | 长期问题 |

## ⚠️ 分析注意事项

1. **推力往往被低估**：员工出于礼貌可能淡化推力，需要从侧面证据推断
2. **拉力有时被放大**：外部机会可能被美化，需要评估信息来源的可靠性
3. **留任力是动态的**：RSU vest完毕、项目完成等时间节点会削弱留任力
4. **核心推力最重要**：找到那1-2个最关键的推力，集中资源解决
5. **挽留窗口有限**：从发现离职意向到做出决策通常只有1-2周
